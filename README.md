### 🏮 Research & Core Projects

#### [Biogas Optimization Research](https://github.com/SarvanshRaj/biogas-optimization-research) 
*Shortlisted for Sakura Science (Japan) | Predictive AI Framework*
- **The Goal:** Optimizing substrate mixtures for 2-stage anaerobic digesters using *Aspergillus oryzae*.
- **The Tech:** Implementing a Supervised Learning framework to predict biogas yield based on protein/carbohydrate ratios.
- **Status:** [Technical Roadmap Phase] - Documenting the predictive framework and sensor-integrated monitoring schematics.

#### [IoT Environmental Monitoring](https://github.com/SarvanshRaj/iot-environmental-monitoring)
*Real-time hardware-software data pipeline*
- **Architecture:** Arduino (DHT11) → Serial Communication → Raspberry Pi 3B+ (Data Logging/Storage).
- **Focus:** High-frequency data capture and environmental variance analysis for localized climate monitoring.

#### [Privacy-Centric Network Infrastructure](https://github.com/SarvanshRaj/privacy-infra)
*Self-Hosting & Digital Sovereignty*
- **The Stack:** Docker, Docker Compose, Tor Bridge, SearXNG.
- **Objective:** Deploying decentralized networking tools on ARM-based hardware to explore private search and network resilience (De-googling).
